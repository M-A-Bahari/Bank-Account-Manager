Overview
This project implements a User Authentication System in Java, featuring user registration and login functionalities. It securely handles user credentials and ensures proper authentication mechanisms.

Features
User Registration: Users can create an account with a username and password.
User Login: Users can log in using their credentials.
Password Hashing: Ensures security using hashing techniques.(This is/ will be implemented on the FileStorage.java class)
Error Handling: Provides meaningful error messages for invalid credentials.
File-Based Storage (Optional): Users' credentials can be stored in a file/database.(We are using file based storage)
